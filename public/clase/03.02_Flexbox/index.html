Layouts con Flexbox

Flexbox (o cajas flexibiles) es una herramienta introducida en CSS3 que nos permite construir layouts (estructuras), alinar y distribuir elementos ocupando el espacio disponible de un contendor de manera sencilla.

La idea principal detrás de Flexbox es la de darle la pósibilidad a un contenedor de distribuir todos sus elementos hijos de manera que puedan trabajar con todo el espacio disponible
