Layouts con Flexbox

Flexbox (o cajas flexibiles) es una herramienta introducida en CSS3 que nos permite construir layouts (estructuras), alinar y distribuir elementos ocupando el espacio disponible de un contendor de manera sencilla.

La idea principal detrás de Flexbox es la de darle la pósibilidad a un contenedor de distribuir todos sus elementos hijos de manera que puedan trabajar con todo el espacio disponible

Documentación: A guide to flexbox

Propiedades de los padres (contenedores)

display

Esto define a un contenedor como flex y afecta a sus hijos directos. A partir de ahora comenzaran a fluir de izquierda a derecha (por defecto)

.flex-container {
    display: flex;
}

direction

Establece el eje principal en el que fluiran los elementos. Los elementos dentro de un contenedor flex fluyen siempre en una dirección que puede ser en modo de row (fila) o column (columna)

.container {
    flex-direction: row | row-reverse | column | column-reverse;
}
  • row: (default) elementos fluyen de izquierda a derecha
  • row-reverse: elementos fluyen de derecha a izquierda
  • column: igual que row, pero de arriba hacia abajo
  • column-reverse: igual a row-reverse pero de abajo a arriba

flex-wrap

Por defecto, todos los elementos flex trataran de adaptarse para permanecer en una sola linea. Este comportamiento se puede cambiar a través de la propiedad flex-wrap permitiendo ajustar como fluyen los elementos

.flex-container {
    flex-wrap: nowrap | wrap | wrap-reverse;
}
  • nowrap: (default) todos los elementos flex estarán en una misma linea
  • wrap: elementos flex se plegaran en multiples lineas si superan el ancho de su contenedor
  • wrap-reverse: Igual a wrap pero en lugar de fluir de izquierda a derecha, lo haran de abajo hacia arriba

justify-content

Esta propiedad define el alineamiento de las cajas en torno al eje principal. Ayuda a distribuir los elementos para ocupar el espacio libre

.flex-container {
    justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
}
  • flex-start: (default) los items se alinean a partir del eje inicial (definido en flex-direction)
  • flex-end: los items se ubican hacia el final de su linea
  • center: los items son centrados en medio de su linea
  • space-between: los items son distribuidos uniformemente en su linea alcanzando a ocupar el 100% del ancho disponible. El primer item se ubica al principio y el último hacia el final de la linea.
  • space-around: los items se distribuyen de manera uniforme en la misma linea pero igualando los espacios a su alrededor.
    Es importante notar que todos los espacios no son visualmente iguales, dado que el primer item presentará una unidad de espacio a su izquierda y dos a su derecha. Esto se debe a que todos los elementos llevan espacios iguales a ambos lados.
  • space-evenly: los items se distribuyen de manera que visualmente los espacios entre ellos sean iguales.

align-items

Muy similar a como justify-content funciona, pero afecta al eje secundario (eje x) en lugar del eje principal

.flex-container {
    align-items: flex-start | flex-end | center | baseline | stretch;
}
  • flex-start: los elementos se ubican a partir del valor 0 del eje secundario
  • flex-end: los elementos se ubican en sentido opuesto al dado por flex-start
  • center: los items se centran en el eje secundario
  • baseline: los elementos se alinean de acuerdo a su linea de base
  • stretch: (default) los elementos se expanden hasta ocupar el 100% de su contenedor (min/max width se respetan)

align-content

Muy similar a como justify-content funciona en cuanto a items, pero en este caso align-items afecta a la linea de elementos

Nota:
Esta propiedad no tiene efecto cuando solo hay una linea de elementos flex

.flex-container {
align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}
  • flex-start: las lineas se ajustan al principio del contenedor
  • flex-end: las lineas se ajustan al final del contenedor
  • center: las lineas se ajustan en el centro del contenedor
  • space-between: las lineas se desplazan ocupando todo el espacio dentro del contenedor pegando las lineas del comienzo y fin a los margenes superior e inferior y distribuyendo el resto en el espacio sobrante.
  • space-around: las lineas se distribuyen uniformemente con igual espacio entre sí
  • stretch: (default) las lineas se estiran para ocupar todo el espacio disponible
