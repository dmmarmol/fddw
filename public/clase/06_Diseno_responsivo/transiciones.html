Clase 06 - Transiciones en CSS3

Previo a la introducción de CSS3, una manera común para animar contenido en una página web era a través de Flash o JavaScript. A medida que CSS3 fue incluyendo propiedades para poder controlar las transiciones, Flash fue cayendo en deshuso y si bien aún es posible generar transiciones con JavaScript, es ideal tanto desde el concepto de una web bien construida como desde la performance que las transiciones sean a través de CSS

transition

La propiedad transition permite definir la transición entre dos estados de un elemento. Hay diferentes estados que pueden ser definidos utilizando pseudo-clases como :hover o :active

La propiedad transition es una propiedad abreviada de transition-property, transition-duration, transition-timing-function, y transition-delay. 

/* Aplicar a 1 propiedad */
/* nombre de la propiedad | duración */
transition: margin-left 4s;

/* nombre de la propiedad | duración | retardo */
transition: margin-left 4s 1s;

/* nombre de la propiedad | duración | función | retardo */
transition: margin-left 4s ease-in-out 1s;

/* Aplicar a 2 propiedades */
transition: margin-left 4s, color 1s;

/* Aplicar a todas las propiedades que cambien */
transition: all 0.5s ease-out;

Nótese que el orden es importante dentro de los elementos de la propiedad. El primer valor que puede ser analizado como tiempo es asignado a la propiedad transition-duration, y el segundo valor que puede ser analizado como tiempo es asignado a la propiedad transition-delay.

Ejemplo con margin-left

.item {
    margin-left: 0;
    /* nombre de la propiedad | duración | retardo */
    transition: margin-left 1s 0.5s;
}
.item:hover {
    margin-left: 2em;
}

transform

La propiedad transform fue también introducida en CSS3 y usándola los elementos pueden ser trasladados, rotados, escalados o sesgados de acuerdo a los valores establecidos.

  1. translateX(x): Permite transicionar un elemento en el eje X
  2. translateY(y): Permite transicionar un elemento en el eje Y
  3. translate(x, y): Abreviatura de translateX() translateY()
  4. translate3d(x, y, z): Similar a translate() pero con el agregado del eje Z como tercer parametro (translateZ())
  5. rotate(deg): Proporcionando un valor en grados (90deg) podremos rotar el elemento en el eje X
  6. scale(value): Funciona y posee las mismas variantes que translate() pero se utiliza para aumentar o reducir el tamaño de un objecto. Sus valor incial es 1 siendo este su escala actual. También existen scaleX(), scaleY() y scaleZ()
  7. skew(): similar a translate y scale pero con ella podremos sesgar un elemento

translate()

.item {
    /* transition sin delay */
    transition: transform 1s;
    transform: translate(0, 0);
}
.item:hover {
    /** Muevase 20px hacia la derecha del eje X */
    transform: translate(20px, 0);
}
.item:active {
    /** Muevase 20px hacia la arriba del eje Y */
    transform: translate(0, -20px);
}

rotate()

.item {
    /* transition sin delay */
    transition: transform 1s;    
}
.item:hover {
    /** Rota 45 grados hacia la derecha */
    transform: rotate(45deg);
}
.item:active {
    /** Valores negativos rotan el elemento hacia la izquierda */
    transform: rotate(-45deg);
}

scale()

.item {
    /* transition sin delay */
    transition: transform 1s;
}
.item:hover {
    /** Dobla su tamaño */
    transform: scale(2);
}
.item:active {
    /** Reduce su tamaño a la mitad */
    transform: scale(0.5);
}

skew()

.item {
    /* transition sin delay */
    transition: transform 1s;
}
.item:hover {
    /** Inclinación hacia la izquierda */
    transform: skew(45deg);
}
.item:active {
    /** Inclinación hacia la derecha */
    transform: skew(-45deg);
}
