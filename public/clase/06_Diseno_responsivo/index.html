Diseño web responsivo

¿Qué es el "Diseño web responsivo"?

En un principio se sabía que en el unico dispositivo en que un usuario podría visualizar un sito web, sería desde una computadora de escritorio. Esto proporcionaba cierta tranquilidad para el desarrollador web dado que solo con asegurarse de que los margenes de su sitio no sebrapasaran los 1024px, se estaría asegurando que una gran variedad de usuarios puedan visualizar su sitio correctamente.

Todo esto cambió con la llegada de los dispositivos mobiles. Infinidad de nuevos tamaños y resoluciones han obligado al desarrollo web a moverse hacia un horizonte más flexbile, que le permita seguir desarrollando una sola vez y que ese mismo desarrollo sea adaptable a la gran variedad de dispositivos disponibles

Es en este momento que surge el concepto de Diseño web responsivo o por sus siglas en ingles RWD (responsive web design) que propone como filosofía "adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas"

Esta tecnología pretende que con un único diseño web, todo se vea correctamente en cualquier dispositivo

Con el paso del tiempo numerosas técnicas de CSS han ido acompañando esta filosofía, como son un mayor uso de unidades relativas (em, rem, %) frente a unidades absolutas (px, pt, cm, etc) y grillas adaptables a través de anchos porcentuales y @media-queries, entre otras

Con una sola versión en HTML y CSS se pueden cubrir todas las resoluciones de pantalla, con lo que el sitio web estará optimizado para distintos dispositivos y resoluciones de pantalla. Esto mejora la experiencia de usuario a diferencia de lo que ocurre, por ejemplo, con sitios web de ancho fijo cuando se acceden desde dispositivos móviles. De esta forma se reducen los costos de creación y mantenimiento cuando el diseño de las pantallas es similar entre dispositivos de distintos tamaños.
