Clase 06 - Media Queries

¿Qué son "media queries"?

@media

Otra tecnologia incorporada en CSS3, los media queries permiten adaptar la representación del contenido a las características del dispositivo. Las media queries dejan que la presentación del contenido se adapte a un rango específico de dispositivos de salida sin tener que cambiar el contenido en sí.

Ver Documentación

Sintaxis

Las Media queries consisten de un media type y una o mas expresiones, implicando características del medio, la cual se resuelve como verdadera o falsa. El resultado de la consulta es verdadera si el tipo de medio especificado en el media query concuerda con el tipo de dispositivo que está siendo mostrado y todas las expresiones en el media query son verdaderas.

Se escriben con el selector @media seguido de un espacio y la(s) condición(es) 

.columna {
    width: 50%;
}
@media (max-width: 1024px) {
    .columna {
        width: 100%;
    }
}

Es posible también dedicar hojas de estilos enteras a un media query aplicando una regla de este tipo a la etiqueta <link> 

<link rel="stylesheet" media="(max-width: 800px)" href="example.css" />

Operadores logicos

Media queries soporta una serie de operadores lógicos

  • ONLY
  • AND
  • Lista separada por comas (OR)
  • NOT

ONLY

El operador only previene que navegadores antiguos que no soportan queries con funciones apliquen los estilos asignados:. 

@media only screen and (max-width: 1024px) { ... }

AND

Permite concatenar condiciones

@media (min-width: 700px) and (orientation: landscape) { ... }

Lista separada por comas (OR)

Las listas separadas por comas se comportan como el operador or cuando es usado en media queries. Cuando utilice una lista separada por comas y alguno de los queries retorna verdadero, el estilo o la hoja de estilos sera aplicada. 

@media (min-width: 700px), (min-height: 800px) { ... }

NOT

El operador not aplica a todo el query y retorna verdadero si es posible y sino retorna falso (como por ejemplo monochrome en un monitor a color o una ventana con un ancho mínimo de min-width: 700px en un monitor de 600px). 

@media not screen and (color), print and (color)
