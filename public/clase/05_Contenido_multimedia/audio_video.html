Formatos de Audio y Video en la web

Etiqueta <audio>

En HTML5 podemos reproducir un archivo de audio a través de la etiqueta <audio>.

De la misma manera que sucede con la etiqueta <img>, es a través del atributo src con el que podemos indicar la ruta del audio que se va a reproducir

Utilizando el atributo controls podremos ver los controles para reproducir y detener el audio como también controlar el volumen. 

<audio controls src="ruta/al/archivo/de/audio.mp3"></audio>

La etiqueta <audio> soporta etiquetas hijas dentro de si. Esto nos permite ubicar un mensaje en caso de que el navegador del usuario no sea compatible con esta etiqueta 

<audio controls src="ruta/al/archivo/de/audio.mp3">
    Tu navegador no soporta la etiqueta <code>audio</code>
</audio>

Etiqueta <video>

El elemento <video> se utiliza para incrustar vídeos en un documento HTML.

De la misma manera que sucede con la etiqueta <audio>, es a través del atributo src con el que podemos indicar la ruta del video que se va a reproducir

Utilizando el atributo controls podremos ver los controles para reproducir y detener el audio como también controlar el volumen. 

<video controls src="ruta/al/archivo/de/video.mp3"></video>

La etiqueta <video> soporta etiquetas hijas dentro de si. Esto nos permite ubicar un mensaje en caso de que el navegador del usuario no sea compatible con esta etiqueta
