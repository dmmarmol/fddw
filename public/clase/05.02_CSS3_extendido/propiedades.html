Bordes redondeados

A través de la propiedad border-radius podemos personalizar el radio de cada esquina de una caja. 

.radius {
    border-radius: 8px; /* 8px en cada esquina */
}
.radius {
    /** 10px arriba en diagonal de arriba hacia abajo y 30px en diagonal de abajo hacia arriba */
    border-radius: 10px 30px; 
}
.radius {
    /** En sentido de las ajugas del reloj comenzando por izq. arriba */
    border-radius: 10px 20px 30px 40px; 
}
.radius {
    border-radius: 50%;
}

Sombras

La propiedad CSS box-shadow describe uno o más efectos de sombra como una lista separada por comas. Permite proyectar una sombra difuminada en la estructura de casi cualquier elemento. Si un border-radius es especificado en un elemento con un box shadow, el box shadow asume las mismas esquinas redondeadas

Valores

Los valores que component a una sombra van separados con espacios y respetan siempre el mismo orden

  1. inset: Si no se especifica (por defecto), la sombra va por fuera de su caja contenedora. Dando la sensación de que la caja se esta despegando del fondo
  2. offset-x & offset-y: definen el desplazamiento de la sombra en el eje X e Y. Los valores positivos de X colocan a la sombra a la derecha del elemento mientras que los negativos a la izquierda. Lo mismo pasa con Y pero abajo y arriba, respectivamente
  3. blur-radius: Cuanto mayor sea este valor, mayor será el difuminado que se aplique a la sombra.
  4. spread-radius: Los valores positivos harán que la sombra se expanda y cresca más. Los valores negativos harán lo opuesto. Si no se especifica, su valor sera 0
  5. color: cualquier valor de color valido para CSS le dará color a la sombra
.shadow {
    box-shadow: 2px 2px 5px 0 black;
}
.shadow {
    box-shadow: 0 5px 2px 0 rgba(0, 0, 0, 0.5);
}

Podemos aplicar estos estilos a través de pseudo-clases para dar feedback al usuario al momento de realizar una acción.

.shadow {
    box-shadow: none;
    transition: box-shadow 400ms ease, margin-top 400ms ease;
}

.shadow:hover {
    box-shadow: 0 5px 2px 0 rgba(0, 0, 0, 0.5);
}

.shadow:active {
    box-shadow: 0 2px 1px 0 rgba(0, 0, 0, 0.2);
    margin-top: 3px; /** 3px = 5px (offset-y) - 2px (box-shadow: offset-y) */
}

:before & :after

Existen dos tipos de pseudo-selectores ::before & ::after que nos permiten modificar el contenido del DOM (HTML) brindandonos la posiblidad de agregar un hijo anterior o posterior al elemento seleccionado

::before

::before crea un pseudo-elemento que es el primer hijo del elemento seleccionado. Es usado normalmente para añadir contenido estético a un elemento, usando la propiedad content. Este elemento se muestra en línea con el texto de forma predeterminada

Solo puede haber un ::before por elemento

.item::before {
    content: '-> ';
    display: inline;
}

::after

::after coincide con el último hijo virtual del elemento seleccionado. Se usa generalmente para añadir contenido estético a un elemento, usando la propiedad CSS content. Este elemento se muestra en línea con el texto de forma predeterminada.

Solo puede haber un ::after por elemento

.item::after {
    content: "Este contenido existe en CSS";
    display: block;
    background-color: yellow;
    font-weight: bold;
    font-size: 12px;
}

Este contenido esta dado por el HTML

:nth-child

La pseudo-clase :nth-child() de CSS coincide con uno o más elementos en función de su posición entre un grupo de hermanos. 

/* Selecciona cada cuarto elementos entre cualquier grupo de hermanos */
:nth-child(4n) {
    background-color: red;
}
