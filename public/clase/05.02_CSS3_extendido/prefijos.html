Prefijos en CSS

Muchas de las nuevas propiedades introducidas en CSS3 no son soportadas por los navegadores actuales, al menos no sin usar prefijos al momento de redactarlas.

Los prefijos en algunas propiedades de CSS vienen a solucionar problemas de compatiblidad existentes entre navegadores. Muchos de estos prefijos son necesarios para que dichas propiedades funcionen en versiones viejas de navegadores

caniuse.com

El sitio caniuse.com nos permite filtrar por nombre de propiedad CSS y nos brinda una gráfica comparativa entre versiones anteriores, actuales y futuras de navegadores web y que soporte le brinda cada navegador a dicha propiedad en cuestion

La captura anterior corresponde una busqueda en caniuse.com sobre la propiedad box-shadow donde podemos ver el soporte de esta propiedad para las versiones del navegador Firefox entre la version 3.5 y 3.6. Sobre el cartel flotante (verde) podemos ver que para afectar a estas versiones es necesario utilizar el prefijo -moz- delante de la propiedad, por lo que su resultado final deberia ser el siguiente 

div {
    box-shadow: 0 1px 2px rgba(0,0,0, 0.5);
    -moz-box-shadow: 0 1px 2px rgba(0,0,0, 0.5);
}

Es necesario incluir la propiedad con y sin prefijo para abarcar la mayor cantidad de navegadores que puedan soportarla tanto de manera nativa (sin prefijo) como con prefijo

Prefijos para Gradientes

Tomando el caso de los gradientes, cuya compatiblidad entre navegadores sigue siendo todo un problema al momento de escribir este articulo, una manera de ampliar el abanico de navegadores soportados es incluyendo la mayor cantidad de prefijos posibles. 

.gradient {                         
    background: -moz-linear-gradient(left,red,orange,yellow, green, blue,indigo,violet);
    background: -webkit-linear-gradient(left,red,orange,yellow, green, blue,indigo,violet);
    background: -ms-linear-gradient(left,red,orange,yellow, green, blue,indigo,violet);
    background: -o-linear-gradient(left,red,orange,yellow, green, blue,indigo,violet);
    background: linear-gradient(to right,red,orange,yellow, green, blue,indigo,violet);                      
}

Sitios web como css3generator.com nos permiten, entre otras propiedades, generar un gradiente a través de una interfaz gráfica facilitandonos la tarea de incluir todos los prefijos necesarios.
