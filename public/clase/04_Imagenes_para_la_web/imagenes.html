Imagenes Para la web

Etiqueta <img> y posibilidades

El elemento de imagen HTML <img> representa una imagen en el documento y es una etiqueta que permite cerrarse en sí misma.

Nota:
Los navegadores no siempre muestran la imagen a la que el elemento hace referencia. Es el caso de los navegadores no gráficos (incluyendo aquellos usados por personas con problemas de visión), sí el usuario elige no mostrar la imagen, o sí el navegador es incapaz de mostrarla porque no es válida o soportada. En ese caso, el navegador la reemplazará con el texto definido en el atributo alt. 

<img src="wrong/path/to/file.jpg" alt="Imagen con cielo azul" />

¿Cómo aplicar estilos a imagenes?

Imagenes redondeadas

img {
    border-radius: 8px
}
img {
    border-radius: 50%
}

Miniatura de imagen

img {
    border: 1px solid #ddd;
    border-radius: 4px;
    padding: 5px;
    width: 150px;
}
img {
    border: 1px solid #ddd;
    border-radius: 4px;
    padding: 5px;
    width: 150px;
}

img:hover {
    outline: 3px solid rgba(0, 140, 186, 0.5);
}

Imagenes responsivas

img {
    max-width: 100%;
    height: auto;
}

Imagenes transparentes

img {
    opacity: 0.25;
}
img {
    opacity: 0.5;
}
img {
    opacity: 1;
}
