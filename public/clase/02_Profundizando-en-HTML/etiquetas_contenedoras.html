Etiquetas Contenedoras

Las etiquetas contenedoras nos permiten agrupar diferentes etiquetas HTML bajo un mismo concepto o criterio. No dicen nada sobre su contenido.

<div>

La etiqueta <div> (de division, o divisor) se usa a modo de contendor y permite diferenciar distintas secciones dentro de un mismo documento

Ejemplo

Titulo incluido en div#1

Este párrafo se encuentra contenido dentro de un <div>

El párrafo siguiente esta próximo al párrafo anterior y aún así dentro del mismo <div>

Titulo incluido en div#2

Este párrafo se encuentra dentro de otro <div>

<table>

La etiqueta <table> se usa para definir una tabla dentro de un documento.
A su vez, para poder definir una tabla integramente se necesitan de otras etiquetas complementarias

Etiquetas complementarias (obligatorias)

  • <tbody> Cuerpo de la tabla
  • <tr> Fila
  • <td> Columna

Ejemplo

Fila 1 - Columna 1 Fila 1 - Columna 2 Fila 1 - Columna 3 Fila 1 - Columna 4
Fila 2 - Columna 1 Fila 2 - Columna 2 Fila 2 - Columna 3 Fila 2 - Columna 4
Nota: Antes de la creación de CSS, los elementos HTML <table> eran frecuentemente utilizados para la disposición (posicionamiento) de una página. Este uso ha sido desalentado desde HTML 4, y el elemento no debe ser usado para dichos fines.
