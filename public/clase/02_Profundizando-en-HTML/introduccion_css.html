Introducción a CSS

CSS deriva del termino "Cascading Style Sheets" u "Hoja de estilos en cascada". Se encarga del diseño y la presentación de una página HTML y de cómo se verán sus elementos en pantalla.

  • Permite dar estilos a un mismo elemento que se repite a lo largo de varias páginas (evita la repetición de código)
  • Se escribe y almacena en archivos con extensión .css y se vincula con un archivo HTML a través de la etiqueta <link>

<link>

La etiqueta <link> sirve para establecer una relación entre el documento actual y un documento externo.

Se utiliza comunmente para incluir hojas de estilo (CSS) dentro de un documento HTML

La etiqueta se abre y se cierra en sí misma y se ubica dentro de la etiqueta <head>

Ejemplo

<head> <link href="../css/style.css" rel="stylesheet" /> </head>
=> Documentación oficial

Selector CSS

Los selectores definen sobre qué elementos HTML se aplicará un conjunto de reglas de CSS.

Su estuctura esta compuesta por un nombre y dentro una o más reglas. Se leen de derecha a izquierda.

Las reglas de CSS siempre se escriben comenzando por el nombre de la propiedad, luego dos puntos (:) y siguiente el valor a asignarle a esa propiedad

Para comenzar, es necesario entender tres tipos de selectores

=> Documentación oficial
  • Selector de "tipo" (etiqueta HTML)
  • Selector de "clase"
  • Selector de "id"

Selector de tipo

p { font-size: 12px; }
=> Documentación oficial

Selector de clase

Se escriben con un punto (.) como prefijo para indicar que el siguiente texto corresponde a una clase

.caja { margin: 0 auto; }
=> Documentación oficial

Selector de id

Se escriben con un punto (.) como prefijo para indicar que el siguiente texto corresponde a un id

#persona-dni-44589762 { margin: 0 auto; }
=> Documentación oficial
