Etiquetas Semánticas

Las siguientes etiquetas semánticas son sólo un grupo de etiquetas básicas que cargan de significado a su contenido. Todas han sido incluidas en el estandar HTML5

<nav>

La etiqueta <nav> sirve para establecer un menú de navegación.

Ejemplo

  • Inicio
  • Etiquetas contenedoras
  • Etiquetas semánticas
=> Documentación oficial

<article>

La etiqueta <article> representa una composición auto-contenida en un documento. Puede ser tanto independiente o reutilizable y debería poder leerse independientemente del resto del sitio.

Ejemplo

Titulo de una nota

Cuerpo de una nota. Puede ser tanto un párrafo como así varios

Aquí otro párrafo más

<aside>

La etiqueta <aside> representa una sección de una página que consiste en contenido que está tangencialmente relacionado con el contenido que le rodea.

Ejemplo

Título de un article dentro de una section

  • Boton #1
  • Boton #2
  • Boton #3
Es posible incluir <article> dentro de <section> o vice versa?
La etiqueta <article> especifica independencia del resto del sitio y es auto-contenida. Por otro lado, la etiqueta <section> define una sección dentro de un documento.
No podemos partir de dichas definiciones para establecer "¿Quien viene primero?", por lo tanto, es probable que en internet encontremos muchos casos en donde la herencia este invertida.
