Propiedades de CSS

Las siguientes propiedades son sólo un grupo de propiedades comunes pueden verse en la gran mayoria de los sitios web.

Ver Fuente

display

La propiedad display tiene muchos valores posibles, pero solo 4 son los más utilizados. 

div {
    display: block;
    display: inline-block;
    display: inline;
    display: none;
}

block: Los elementos del tipo block ocupan todo el espacio disponible y no pueden ser ubicados en la misma linea horizontal que otro elemento del tipo block (excepto que posea la propiedad float). Es el valor por default de elementos como los <div>, <p>, <ul> o <section>, entre otros.

inline: Los elementos del tipo inline pueden ser ubicados en la misma linea horizontal que otro elemento del tipo inline y se ajustan al espacio ocupado por el contenido que poseen.
La propiedad float no tiene ningún efecto sobre ellos.
Es el valor por default de elementos como <a>, <span> o <strong>, entre otros.

inline-block: Los elementos del tipo inline-block combinan las propiedades de los tipo block e inline block. Es posible determinar medidas de ancho y alto al mismo tiempo que permite tener otros elementos del tipo inline-block e inline en su misma linea horizontal.
La propiedad float no tiene ningún efecto sobre ellos.

none: Este valor permite ocultar el elemento dentro del sitio de manera que no solo no será visible sino que tampoco ocupara espacio (al contrario de lo que ocurre con la propiedad opacity).

width & height

Las propiedades width y height nos permiten establecer las medidas de ancho y alto de un elemento 

div {
    width: 50%;
    height: 100%;
}

Posibles valores

  • px: Pixels
  • em: Unidad de medida relativa al valor de la propiedad font-size
  • rem: Muy similar a em, pero toma el valor de font-size establecido en la etiqueta html
  • %: Porcentajes
  • auto: automático

Existen tambien variantes para ancho/alto maximo/minimo como son:

div {
    min-width: 300px;
    max-width: 80%;
    min-height: 300px;
    max-height: 100%;
}

margin & padding

Las propiedades margin y padding afectan los espacios por fuera y por dentro de un elemento, respectivamente. 

div {
    margin-top: 10px;
    margin-bottom: 10px;
    padding-left: 30px;
    padding-right: 30px;
}

Originalmente, los margenes/paddings se escriben de la siguiente manera

div {
    margin-top: 10px; /** Superior */
    margin-right: 10px; /** Derecha */
    margin-bottom: 10px; /** Inferior */
    margin-left: 10px; /** Izquierda */            
}

Abreviación

En ambos casos, pueden escribirse también en su manera corta

div {
    margin: 10px; /** 10px a todos los ejes */    
    margin: 10px 20px; /** 10px en eje vertical y 20px en horizontal */
    margin: 10px 20px 10px 20px; /** Igual al caso anterior */
}

Comparten la posibilidad de usar las mismas medidas que width & height

Centrado Horizontal

Margenes horizontales con valor auto sirven para centrar contenido (que posea width/max-width) horizontalmente 

div {
            margin: 0 auto;
        }

border

La propiedad border afectan los bordes de la caja de un elemento.

Esta compuesta por 3 tipos de propiedades 

div {
    border-width: 3px; /** Grosor del borde */
    border-style: solid; /** Estilo del borde (normalmente su valor es 'solid') */
    border-color: red; /** Color del borde */
}

Abreviación

Podemos también redactar esta propiedad en una sola linea

div {
    border: 3px solid red;
}

float

La propiedad float en escencia, sirve para acercar el eje mas lejano del elemento afectado junto al eje más cercano del elemento siguiente.
En otras palabras, anula el efecto de los elementos del tipo block y permite juntar dos o más elementos en una mísma linea

Puede llevar 3 tipos de valores 

div {
    /** Empuja el elemento a la izquierda y los siguientes a su lado derecho */
    float: left; 
    /** Igual a `left`, pero en sentido opuesto */
    float: right;
    /** Remueve cualquier valor anterior de `float` */
    float: none;
}
Ver Documentación Ejemplos

clear

Si bien float puede sernos muy útil, también puede traernos dolores de cabeza. Para evitarlo, hay que "cancelar" los floats y para ello surge la propiedad clear

La propiedad clear suele aparecer con estos 3 valores 

div {
    clear: left; /** Cancela floats del lado izquierdo del elemento */
    clear: right; /** Igual a `left`, pero en sentido opuesto */
    clear: both; /** Cancela floats a ambos lados del elemento */
}

background

Esta propiedad sirve para editar el fondo de una caja, ya sea como un color, un gradiente o una imagen 

div {
    background: red; /** valor por nombre */
    background: #F00; /** valor hexadecimal abreviado */
    background: #FF0000; /** valor hexadecimal completo */
    background: rgb(255, 0, 0); /** valor rgb */
    background: rgba(255, 0, 0, 0.5); /** valor rgb + transparencia */
    /** ruta hacia una imagen relativa al archivo .css */
    background: url('images/imagen.jpg'); 
}

Los ejemplos anteriores de brackground utilizan su sintáxis corta. La manera más detallada de escribirlas sería: 

div {
    background-color: #F00;
    background-image: url('images/imagen.jpg');
}

font

La propiedad font afecta la apariencia del texto en un sitio web

Abreviación

p {
    font: italic 9px/12px "Helvetica", sans-serif;
}

Desgloce

p {
    font-family: "Helvetica", sans-serif;
    font-size: 9px;
    line-height: 12px;
    font-style: 9px;
}
