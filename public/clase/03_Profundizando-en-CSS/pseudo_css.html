Pseudo Clases y Pseudo Elementos

Dijimos que un selector puede ser de tipo, de clases y de id pero también existen las pseudo-clases y pseudo-elementos que nos permiten ser un poco más granular al momento de aplicar estilos

A diferencia de los selectores del tipo clase o id, estos no pueden llevar un nombre personalizado. Las Pseudo clases se escriben con el prefijo : (dos puntos). Ejemplo :active
Los Pseudo elementos se escriben con doble dos puntos :: para diferenciarse.

Pseudo Clases

Las pseudo clases afectan a elementos del DOM pero solo dependiendo de su estado en lugar de en su información. Existen algunas pseudo-clases muy comunes

  • :hover
  • :active
  • :focus
  • :first-child
  • :last-child

Hover

Los estilos presentes se verán cuando el usuario posa el mouse sobre el elemento

p {
        font-size: 12px;
    }
    p:hover {
        font-size: 24px; /** El tamaño de fuente se duplica */
    }
Pasa el mouse sobre mi

Active

Los estilos presentes se verán cuando el usuario posa haga click (y no lo suelte) sobre el elemento

p {
        background: transparent;
    }
    p:active {
        color: white;
        background: red; /** El fondo pasa de transparente a ser rojo */
    }
Haz click sobre mi

Focus

Los estilos presentes se verán cuando el usuario haga foco sobre el elemento

input {
    border: none;
}
input:focus {
    border: 1px solid green;
}

First & Last child

En cada uno los estilos afectaran al primer/último elemento hijo, respectivamente

p {
    background: violet;
}
p:first-child {
    background: green;
}
p:last-child {
    background: cyan;
}

Pseudo Elementos

A diferencia de las pseudo clases, los pseudo elementos crean nuevos elementos que no existen en el HTML al momento de escribirlo y permiten ser manipulados para aplicar estilos evitando incrementar el tamaño del código HTML

  • ::first-line
  • ::first-letter

First line y First Letter

Funcionan muy parecidos a :first-child pero en este caso, el elemento a estilizar es la primera linea de un texto o la primera letra 

.capitular::first-letter {
    font-size: 24px;
}
p.destacado::first-line {
    color: blue;
}
