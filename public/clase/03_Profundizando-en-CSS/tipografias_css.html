Tipografías para la web

Si bien podemos utilizar las fuentes instaladas en el dispositivo del usuario, también es posible incoporar familias tipográficas personalizadas a nuestro sitio web.

Selector @font-face

@font-face permite al desarrollador especificar fuentes online para visualizar en sus páginas web. Al permitir a los desarrolladores proporcionar sus propias fuentes, @font-face elimina la necesidad de depender de fuentes de usuarios instaladas en sus computadoras.

Sintaxsis

@font-face {
    font-family: "Open Sans", "Roboto", "Helvetica", sans-serif;
    src: '../font/open-sans.ttf'; /** Ruta del archivo de la fuente */
    /** 
    * Algunas propiedades opcionales
    */
    font-weight: 500; /** Peso tipográfico (500 = regular) */
    font-style: normal; /** Estilo (variable) */
}

Columnas tipográficas

Gracias a CSS3, es posible dividir el contenido en Columnas sin necesidad de generar más HTML

Sintaxsis

.columns {
    columns: 3 12px; /** Version Corta */
    columns-count: 3; /** Cantidad de columnas */
    columns-gap: 3; /** Cantidad de columnas */    
}

Alineación tipográfica

Al igual que en una pieza editorial, podemos alinear el texto hacia un eje, centrarlo o justificarlo

Sintaxsis

.align-left { text-align: left; }
.align-right { text-align: right; }
.align-center { text-align: center; }
.align-justify { text-align: justify; }
